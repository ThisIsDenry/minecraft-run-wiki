

# Launchers

## Default Launcher - Single Profile

> **pros:**
> * easy to set up
> * small file usage

> **cons:**
> * world file buildup
> * speedrun settings all the time
> * mod's need to be manualy changed when other mods are wanted

## Default Launcher - Multi Profile

> **pros:**
* run world files isolated from other instances of minecraft
> * speedrun settings only on runs
> * mods are static

> **cons:**
> * larger file usage
> * medium amount of setup

## MultiMC

> **pros:**
> * small file usage
> * run world files isolated from other instances of minecraft
> * speedrun settings only on runs
> * mods are static

> **cons:**
> * third party launcher
> * medium amount of setup

# Mods

## Vanilla

> **pros:**
> * no setup

> **cons:**
> * slower running game
> * planer fog only on a few systems

## CaffeineMC

> **pros:**
> * faster running game
> * planer fog

> **cons:**
> * setup required

## Optifine
> **pros:**
> * faster running game

> **cons:**
> * any run using optifine will be rejected

# Optinal things you should do

### **Gama**
> makes dark areas brighter. To enable open `.minecraft/options.txt` and gamma to 5.0
### **Subtitles**
> Lets you see sounds that would be to quiet to hear otherwise and lets you see the direction a sound is from
### **Planer fog**
> Lets you see further in the nether
