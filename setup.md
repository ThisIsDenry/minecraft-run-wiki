# Launchers

## Default Launcher - Single Profile

> **pros:**
> * easy to set up
> * small file usage

> **cons:**
> * world file buildup
> * speedrun settings all the time
> * mod's need to be manualy changed when other mods are wanted

### *Setup*
> 1. Download and install minecraft
> 3. open minecraft
> 2. login
> 4. open `Instalations`
> 5. click edit on target profile
> 6. set `version`
> 7. save
> 8. launch game
> 9. *(optional)* install mods

## Default Launcher - Multi Profile

> **pros:**
* run world files isolated from other instances of minecraft
> * speedrun settings only on runs
> * mods are static

> **cons:**
> * larger file usage
> * medium amount of setup

### *Setup*
> 1. download and install minecraft
> 2. open minecraft
> 3. login
> 4. open `Instalations`
> 5. click `New`
> 6. set `version`
> 7. change game directory to be something other then the default (when refering to `.minecraft` folder from here on this is the directory being referred to)
> 8. save
> 9. set active profile to the new profile
> 10. launch game
> 11. *(optional)* install mods

## MultiMC

> **pros:**
> * small file usage
> * run world files isolated from other instances of minecraft
> * speedrun settings only on runs
> * mods are static

> **cons:**
> * third party launcher
> * medium amount of setup

### *Setup*
> 1. download and install multiMC
> 2. login
> 3. open multiMC
> 4. click `Add Instance`
> 5. select target vesion
> 6. launch new instance
> 7. *(optional)* install mods

# Mods

## Vanilla

> **pros:**
> * no setup

> **cons:**
> * slower running game
> * planer fog only on a few systems

## CaffeineMC

> **pros:**
> * faster running game
> * planer fog

> **cons:**
> * setup required

WARNING: do not install Fabric API as it is not needed for CaffeineMC and will get your run rejected

### *Setup - Default Launcher*
> 1. download [fabric launcher](https://fabricmc.net/use/)
> 2. run the installer for your target version of the game
> 3. set version of game in launcher to be fabric-loader-xxx
> 4. download [sodium]( https://mrmangohands.github.io/sodium-redirect/), [lithuim](https://mrmangohands.github.io/lithium-redirect/) and, [phosphor](https://mrmangohands.github.io/phosphor-redirect/) (links are for 1.16.1)
> 5. add the mods to `.minecraft/mods` folder 

### *Setup - MultiMC*
> 1. right click profile and `edit instance`
> 2. click install forge
> 3. click `Open .minecraft'
> 4. download [sodium]( https://mrmangohands.github.io/sodium-redirect/), [lithuim](https://mrmangohands.github.io/lithium-redirect/) and, [phosphor](https://mrmangohands.github.io/phosphor-redirect/) (links are for 1.16.1)
> 5. add the mods to `.minecraft/mods` folder

# Optinal things you should do

### **Gama**
> makes dark areas brighter. To enable open `.minecraft/options.txt` and gamma to 5.0
### **Subtitles**
> Lets you see sounds that would be to quiet to hear otherwise and lets you see the direction a sound is from. To enable open `Options > Accessibility Settings... > Show Subtitles`
### **Planer fog** 
> Lets you see further in the nether *only with mangohands soduim. To enable open `Options > Video Settings... > Unofficial > Use Planar Fog`
